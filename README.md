# A 股选股工具

基于 **DearPyGui**(Dear ImGui 的 Python 封装)开发的桌面版 A 股选股软件。
支持多策略选股、大盘趋势过滤、历史回测、参数寻优、基本面筛选、实时盯盘预警、
选股报告导出等全流程能力。

## 目录结构

```
选股/

├── requirements.txt        依赖清单
├── 启动.bat                Windows 一键启动
```

## 三层架构

- **数据层**:用 akshare(免费、无需 token)拉取全市场行情/基本面/实时快照,存本地 SQLite,避免重复联网。数据源限流时"能拉多少算多少 + 缓存",拉不到留空不阻塞选股。
- **策略层**:指标计算 + 可插拔策略 + 大盘过滤 + 全市场打分排序 + 回测 + 参数寻优。这是软件的核心价值。
- **界面层**:DearPyGui 渲染,左侧配置策略,右侧看结果、K 线、回测、盯盘。

## 核心功能

- **多策略选股**:8 个内置策略(多因子综合/均值回归/潜力股/MACD金叉/放量突破/回踩均线/量价齐升/超跌反弹),参数可在界面滑块调节。
- **大盘趋势过滤**:大盘走弱的交易日不开仓,抬胜率、砍回撤。
- **行业分类 + 多因子选股**:全 A 股扩池,支持行业维度。
- **基本面筛选**:选股结果的后置过滤器,按 PE/PB/ROE/市值上下限剔除,阈值为 0 即不启用该项。数据走百度估值(PE/PB/市值)+ 巨潮(ROE)。
- **历史回测**:无未来函数,逐日切片喂策略,输出收益/胜率/回撤/盈亏比 + 资金曲线 + 交易明细。
- **参数寻优**(两类):
  - *卖出参数网格调优*:遍历止盈/止损/持有天数(27 组),固定买入逻辑。
  - *买入参数寻优*:遍历策略自身声明的关键买入参数(`cls.params`,默认每参数 4 档、最多 3 个参数),固定卖出参数,找出样本最优买入信号。
- **实时盯盘 + 价格预警**:新浪全市场快照(一次拉全市场),自选盈亏实时刷新;可对个股设价格上下限/涨跌幅上下限,触发时高亮提示。
- **多周期 K 线 + 当日分时**:K 线区顶部「日 / 周 / 月 / 分时」切换。周/月线由日线本地聚合(开=首/高=max/低=min/收=末/量额=求和),无需额外拉数据,均线/MACD 自动按新周期重算;「分时」画当日分时曲线(价格线涨红跌绿 + 均价黄线 + 成交量),并**后台每 5 秒自动刷新,仅拉当前正在查看的这一只**(免费源限流友好)。分时数据走新浪 `stock_zh_a_minute`(1 分钟线,股票/ETF 均可)。
- **ETF 数据**:主流宽基 + 热门行业 ETF(约 50 只白名单,沪深300/中证500/科创50/半导体/医药/黄金/纳指等)。ETF 日线与股票**共用 daily_kline 表**,格式一致,因此**策略选股/回测/K线/参数寻优全部自动支持 ETF**;另有「ETF榜单」标签页显示实时行情(按涨跌幅/成交额/最新价排序),可加自选、看 K 线。数据走新浪 `fund_etf_category_sina`(列表+快照)+ `fund_etf_hist_sina`(日线)。
- **选股报告导出**:一键导出 Excel(5 sheet:概览/策略排行榜/今日推荐/选股结果/自选持仓)和自包含 HTML,涨红跌绿,文件落在 `reports/`。

## 使用流程

1. 点「更新股票数据」——拉取股票列表 + 日线;勾选「同步基本面」可增量补 PE/PB/ROE/市值;勾选「同时拉取ETF」可增量补约 50 只主流 ETF。
2. 选择策略,调节参数滑块;需要时开启基本面过滤。
3. 点「开始选股」——扫描本地缓存,命中结果显示在右侧表格(含 PE/PB/ROE%/市值列)。
4. 点表格里的股票代码,下方绘制其 K 线(A 股习惯:涨红跌绿);用 K 线区顶部「日/周/月/分时」切换周期,分时会自动每 5 秒刷新当前这只。
5. 回测面板可对策略做历史回测、卖出参数网格调优、买入参数寻优。
6. 「自选持仓/盯盘」标签页可刷新实时行情、管理价格预警。
7. 「导出报告」把当前排行榜/推荐/选股结果/持仓打包成 Excel + HTML。

## 如何添加你自己的策略

打开 `app/strategy/base.py`,照葫芦画瓢:

```python
class MyStrategy(BaseStrategy):
    name = "我的策略"
    desc = "策略说明"
    params = [
        Param("threshold", "阈值", 默认值, 最小, 最大),
    ]

    def evaluate(self, df):
        # df 已含 ma5/ma20/rsi14/boll_up 等指标列,最后一行是最新
        row = df.iloc[-1]
        selected = row["close"] > row["ma20"]   # 你的选股条件
        score = row["close"]                     # 打分(用于排序)
        return selected, float(score), "命中原因"

# 注册它
ALL_STRATEGIES = {
    ...,
    "my_strategy": MyStrategy,
}
```

参数会自动在界面上生成滑块,并自动纳入「买入参数寻优」的搜索空间,无需改 UI 代码。
